# Advent of Code 2025

This repository will (hopefully) contain a solution to the [`Advent of Code 2025`](https://adventofcode.com/2025) at some point.

## Running a solution

This project uses [`uv`](https://github.com/astral-sh/uv) for dependency management and Python execution.

To run a specific day's solution:

```bash
uv run dayXX.py path-to-your-input-file
